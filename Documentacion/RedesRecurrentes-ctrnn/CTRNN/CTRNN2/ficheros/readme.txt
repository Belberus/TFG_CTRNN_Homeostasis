Para cambiar entre el entrenamiento del experimento 1 y 2-3 comentar/descomentar las líneas correspondientes del script GeneticAlgorithm1.
Para entrenar ejecutar 'GeneticAlgorithm1' (tras comentar/descomentar lo que se requiera).
Una vez entrenado el resultado está almacenado en la variable x.

Para ejecutar el experimento 1 lanzar 'OrientationEvolve(x)'
Para lanzar el experimento 2 lanzar 'ObjectDiscriminateInterNeuronsEvolve(x)'
Para lanzzar el experimento 3 (se entrena con el mismo script que el 2) lanzar 'DualAgentSystem(x)'

Si se necesita cambiar el rango de valores hacerlo sobre la variable 'X_start' (preferiblemente después del entrenamiento, pues influye en éste).
